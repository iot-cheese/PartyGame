# クイックスタートガイド

このドキュメントでは、PartyGameアプリを最速で起動する方法を説明します。

## 🚀 5分でスタート

### 前提条件
- Mac (macOS 13.0+)
- Xcode 14.0+

### ステップ1: プロジェクトを開く
```bash
open PartyGame.xcodeproj
```

### ステップ2: シミュレータを選択
Xcodeのツールバーから「iPhone 14 Pro」などのシミュレータを選択

### ステップ3: 実行
キーボードで `⌘ + R` を押す、または「Run」ボタンをクリック

## 🎮 アプリの使い方

### メインメニュー
アプリを起動すると、3つのゲームが表示されます：
- 🎲 真実か挑戦か
- 🔢 ランダム選択
- 👥 チーム分け

### ゲームを選択
好きなゲームをタップして開始！

## 📱 主な機能

### 真実か挑戦か
1. 「真実」または「挑戦」ボタンをタップ
2. ランダムな質問やタスクが表示される
3. 楽しんで！

### ランダム選択
1. 最小値と最大値を入力
2. 「ランダム生成」ボタンをタップ
3. 結果が表示される

### チーム分け
1. 参加者の名前を入力（1行に1人）
2. チーム数を選択
3. 「チーム分け開始」ボタンをタップ
4. 結果を確認

## 🛠️ カスタマイズ

## ❓ トラブルシューティング

### Xcodeが見つからない
```bash
xcode-select --install
```

### シミュレータが起動しない
Xcodeメニュー > Preferences > Locations でCommand Line Toolsを確認

### ビルドエラー
1. `⌘ + Shift + K` でクリーン
2. 再度 `⌘ + R` でビルド
